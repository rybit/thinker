# thinker

janky tool for just querying rethink db

```
Usage:
  thinker <db> <table> <index> <id> [flags]

Flags:
  -c, --config string   a config file to use
  -d, --debug           enable debug logging
  -f, --follow          if we should follow changes
  -h, --help            help for thinker
  -H, --host string     host to use for rethink (default "localhost")
  -k, --key string      the auth key to use when connecting
  -p, --port int        port to use for rethink (default 28015)
  ```
